# HOW TO EDIT
Open the whitelist.txt file, press the pencil on the right that says 'Edit this File'

To add a faction, add the factions name by exact, followed by the groups ID. For example,
to add "TITAN | Military Corporation", you would write:

TITAN | Military Corporation=6209818

Be mindful that this is __case sensitive__.

To add commits, add a hashtag symbol before the designation.
